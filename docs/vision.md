# Pensieve 愿景

## 终极使命

渐进式达到完全自动化。

## 自动化程度定义

一次 Loop 能完成的 task 数量：

- 10 个以内：低自动化
- 100 个：完全自动化

每次 Loop 结束时，统计当前 task 完成情况，用于评估自动化程度。

## 渐进的含义

初期自动化程度很低，但通过 Agent 的自改进，能代替你回答的问题会越来越多。

你的每一次对话，都会被 link-haiku 视作一次自改进的机会。

最终：
- 你的 Loop 能一次性完成的 task 数量越来越多
- 主题不再容易漂移
- 准则越来越清晰

## self-improve Agent

一个专门用于自改进的 Agent。

它是最了解这个插件运作逻辑的。当收到改进要求后，它会自己判断如何最符合整体利益地进行改造，而不是只听从要求。
